## Notes

- **ADR-055 KG Sync** — Syncs the Knowledge Graph with the `on-touch` (default) or `full-backfill` mode.
  Full-backfill is an ops-only operation and should not be scheduled.
- **AGW Bootstrap / K8s Bootstrap** — One-shot bootstrap workflows for AgentGateway infrastructure.
  They are idempotent but slow; run only when provisioning a new node.
- **Agent-Fix Chain** — Autonomous agent loop for diagnosing and fixing issues; requires `task`,
  `context_uuid`, and optionally `max_iterations`.
- Workflows with no dispatch inputs run on `branch + environment` defaults only.
